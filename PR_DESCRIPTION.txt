## Project Summary

- **Project name**: Ball Catch - 接住滚动的球
- **Robot platform**: Custom MuJoCo five-finger dexterous hand (21 DOF)
- **Task goal**: Dynamic ball-catching with real-time hand tracking and multi-finger coordination
- **Technical approach**:
  - Adaptive gain PID controller with velocity prediction
  - Time-horizon based ball trajectory extrapolation
  - Grasp timing prediction algorithm
  - Multi-finger tactile feedback integration
- **Core features**:
  - Real-time hand tracking with error-based adaptive gain (2.0x/1.5x/1.0x/0.9x)
  - Ball velocity prediction using 0.5s time horizon
  - 5-channel tactile sensor visualization
  - Dual-camera view (front + top) with HUD overlay
  - Deterministic stress testing with 32 seeds
- **Highlights**:
  - 80/100 Grade A performance
  - 4-finger stable grasp achievement
  - 100% success rate on medium/hard difficulties
  - All 8 task gates verified
- **Current limitations**:
  - Tracking error ~8cm (limited by hand damping)
  - Single ball trajectory
  - Response time constrained by physics simulation
- **Future improvements**:
  - Model predictive control for <5cm tracking error
  - Multiple trajectory directions
  - Full 5-finger coordinated catch

## How to Run

```bash
# Single episode with video
python run_ball_catch.py --seed 42 --difficulty easy

# Stress testing (32 trials)
python ball_stress_test.py --seeds 32 --difficulty easy

# Generate analysis reports
python ball_summary_report.py --output outputs --save
python ball_response_time_analysis.py --output outputs --save
python ball_force_analysis.py --output outputs --save
```
